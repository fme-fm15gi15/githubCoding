Copilot + VS Code 開發挑戰

透過這 9 個系統，從基礎 CRUD 進化到 AI 整合與系統架構設計。

🧾 進階核心

自動化發票辨識記帳系統

使用 OCR 技術自動提取發票數據並存入資料庫。

核心功能

影像預處理、Tesseract OCR/Google Vision API 串接、Regex 數據清洗。

Copilot 練習點：

讓 Copilot 生成處理複雜發票格式的正規表示式 (Regex)，並協助撰寫影像轉灰階的代碼。

🧋

智慧辦公室飲料訂購系統

解決公司內部揪團痛點，自動計算金額與提醒。

核心功能

多店同時訂購、自動分帳、Line Notify 提醒、熱門品項統計。

Copilot 練習點：

使用 @workspace 詢問如何設計多對多的資料庫 Schema (User/Order/Item)。

訂購系統
📷

共享器材租賃管理平台

攝影器材或空間預約，整合 IoT 電子鎖。

核心功能

即時日曆預約、自動租約生成、押金管理、損毀報修追蹤。

Copilot 練習點：

練習生成複雜的日曆 UI 邏輯與防止重複預約的後端檢查程序。

租賃平台
🥗

個人化健康餐飲與卡路里

自動食譜推薦與目標追蹤系統。

核心功能

拍照辨識食物營養、目標體重設定、食譜與店家 API 串接。

Copilot 練習點：

練習使用 Copilot 生成用於資料分析的計算演算法 (BMR/TDEE)。

健康餐飲系統
🔧

專業職人媒合評價系統

水電、清潔等服務的地域性媒合平台。

核心功能

地圖定位服務、雙向評價體系、即時聊天室、訂金託管。

Copilot 練習點：

詢問 @workspace 如何整合 Google Maps API 進行經緯度距離計算。

媒合平台
👕

虛擬穿搭建議工具

結合氣象與 AI 的智慧衣櫥建議。

核心功能

衣物照片上傳、天氣 API 串接、根據氣溫生成的 AI 建議。

Copilot 練習點：

讓 Copilot 編寫向 LLM API 發送的 Prompt Template (如：根據 15度 推薦穿搭)。

穿搭建議系統
📚

二手書與筆記交易平台

針對校園市場的數位筆記保護與交易系統。

核心功能

ISBN 掃描自動帶入資訊、浮水印保護技術、系所分類搜尋。

Copilot 練習點：

練習實作 PDF 預覽與動態浮水印添加的 Canvas 代碼。

🏢

智慧社區物聯管理系統

解決社區溝通與繳費不便的數位化工具。

核心功能

數位投票系統、公設預約、包裹通知、管理費線上支付。

Copilot 練習點：

詢問如何設計一個具備防弊機率的匿名投票統計邏輯。

社區管理系統
🐱

寵物健康管理與異狀監測

記錄生活數據並提供異常報警功能。

核心功能

項圈數據整合、飲食排泄記錄、獸醫預約、異常警報。

Copilot 練習點：

讓 Copilot 用 Chart.js 寫出動態顯示寵物活動量趨勢圖表的元件。
